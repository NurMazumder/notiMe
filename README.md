# NotiMe README

## Overview

NotiMe is a web application that scrapes data from [AsuraScans](https://asuracomic.net) to get the latest Manga updates. Users can save their favorite Manga to their bookmarks and receive notifications when a new chapter is released.

## Features

- **Latest Manga Updates**: Get the most recent Manga updates from AsuraScans.
- **Bookmark Management**: Save your favorite Manga to your bookmarks for easy access.
- **Chapter Notifications**: Receive updates when new chapters of your bookmarked Manga are released.
